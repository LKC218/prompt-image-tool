# PC 首页搜索栏移除描边提示计划

> 版本：v1.0.0
> 创建日期：2026-07-09
> 范围：PC 端首页搜索栏移除所有 `border` 描边（常驻浅色边 + 聚焦 accent 高亮边），改为纯新拟态阴影表现
> 关联计划：`PC首页搜索栏新拟态复刻计划-260709.md`

## 一、思考与分析

### 1.1 技能与上下文嗅探

- 根目录未发现 `SKILL.md`。
- 本次为「视觉减法」微调：在上一个新拟态复刻计划落地的基础上，移除搜索栏的 `border` 描边，使凸起完全由双向 `box-shadow` 承担。
- 当前结构：`src/js/pc-home.js`（`pcHomeSearch > __outer > __inner` 双层）；样式：`src/css/pc.css` 6007 起 `.pc-home-search-bar` 区块。

### 1.2 当前描边来源（已确认，用户选择「两者都移除」）

1. **常驻描边**：`.pc-home-search-bar { border: 1px solid rgba(255, 255, 255, 0.6); }`（第 6008 行）—— 新拟态外框的浅色边。
2. **聚焦描边提示**：`.pc-home-search-bar:focus-within { border-color: color-mix(in srgb, var(--pc-accent) 40%, transparent); }`（第 6016-6018 行）—— 点击输入框时外框变 accent 半透明高亮边框。

### 1.3 目标

- 删除上述两处 `border` 相关声明，搜索栏凸起与聚焦反馈**纯靠新拟态 `box-shadow`**（外框双向阴影 + `__outer` 内凹 `inset` 阴影）。
- 保留聚焦时**图标上浮 + 变色**（`acc`ent，第 6045-6048 行 `.pc-home-search-bar:focus-within .pc-search-icon`）—— 该效果属于图标反馈，不在「描边提示」移除范围内。
- 保留轻量旋转反馈（`is-searching`）与 `prefers-reduced-motion` 降级，不受影响。

### 1.4 风险与约束

- 移除 `border` 后若项目未全局 `box-sizing: border-box`，外框尺寸会微增 2px（高/宽各 +1px）。现代 reset 通常已设 `border-box`，但验收需确认无布局跳动。
- 全局类 `.pc-search-icon`/`.pc-search-input`/`.pc-home-inline-icon` 不受影响，改动仅限 `.pc-home-search-bar` scope。

## 二、实施计划

### Phase 1：删除常驻描边

修改 `src/css/pc.css` 第 6007-6014 行 `.pc-home-search-bar` 块：

```css
/* 移除前 */
.pc-home-search-bar {
    min-height: 62px;
    display: flex;
    align-items: center;
    margin: -18px 0 2px;
    padding: 8px;
    position: relative;
    z-index: 2;
    background: var(--pc-surface);
    border: 1px solid rgba(255, 255, 255, 0.6);   /* ← 删除此行 */
    border-radius: 22px;
    box-shadow:
        9px 9px 18px rgba(120, 90, 50, 0.14),
        -9px -9px 18px rgba(255, 255, 255, 0.95);
    transition: border-color 0.2s ease, box-shadow 0.25s ease;  /* ← 去掉 border-color */
}

/* 移除后 */
.pc-home-search-bar {
    min-height: 62px;
    display: flex;
    align-items: center;
    margin: -18px 0 2px;
    padding: 8px;

}
```

### Phase 2：删除聚焦描边

修改 `src/css/pc.css` 第 6016-6018 行，**整块删除** `.pc-home-search-bar:focus-within` 的 border-color 规则（注意保留后续第 6045 行的 `.pc-home-search-bar:focus-within .pc-search-icon` 图标规则）：

```css
/* 删除整块 */
.pc-home-search-bar:focus-within {
    border-color: color-mix(in srgb, var(--pc-accent) 40%, transparent);
}
```

删除后，聚焦时外框不再有描边变化，仅图标保持上浮 + 变色反馈。
