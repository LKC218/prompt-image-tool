# 发布页介绍与安卓安装包校验计划

## 思考与分析

本次目标分为两个部分：一是将 GitHub Release 的说明从简短更新摘要升级为正式的软件功能与特色介绍，二是核对 Android 安装包是否真实存在于 Release 附件区，并在缺失时补传。

当前已核对到的事实：

- 发布版本：`v2.3.1`
- 发布页：`https://github.com/LKC218/prompt-image-tool/releases/tag/v2.3.1`
- 本地安装包存在：
  - `releases/PromptImageManager-Setup-2.3.1.exe`
  - `releases/PromptImageManager-v2.3.1-Android.apk`
- 远端 GitHub Release API 当前返回 2 个附件：
  - `PromptImageManager-Setup-2.3.1.exe`
  - `PromptImageManager-v2.3.1-Android.apk`

注意：Release 正文中的文件名只是说明文本，不等于 GitHub 页面底部的附件下载区。若页面底部没有显示 APK，仍需以 GitHub API 和实际下载链接为准做二次校验。

## 实施计划

### 1. 发布资产校验

1. 使用 GitHub API 读取 `v2.3.1` Release。
2. 校验附件数量、文件名、文件大小和下载链接。
3. 对 APK 下载链接做 `HEAD` 或小流量请求，确认返回状态可用。
4. 若 APK 附件缺失，删除残缺同名资产后重新上传本地 APK。
5. 重新读取 Release 附件列表，确认 Windows 与 Android 两个安装包均存在。

### 2. 发布页文案重写

将 Release 正文调整为面向用户的介绍结构：

1. 软件简介：说明 PromptImageManager 是提示词、图片与版本管理工具。
2. 适用场景：整理生图提示词、管理配图、跨设备同步、离线本地使用。
3. 核心功能：提示词库、分类标签、图片管理、版本记录、导入导出、局域网同步。
4. 版本亮点：PC 与 Android 互通、动态端口、同步方向选择、图片查看器、桌面安装包体验。
5. 安装包下载：明确 Windows 与 Android 下载项。
6. 校验信息：保留 SHA256。
7. 使用提醒：Android 首次安装可能需要允许安装未知来源应用。

### 3. 文案编码策略

由于上一次 PowerShell 管道写入 Release 正文时出现过中文乱码风险，本次正文更新采用以下策略：

1. 优先使用 Python 直接以 UTF-8 调用 GitHub API。
2. 不通过 PowerShell 拼接或转码大段中文正文。
3. 更新后立即读取 GitHub API 返回的正文，确认中文未被替换为问号。
4. 再打开发布页人工核对正文排版和附件区。

### 4. 发布后核对

1. 确认 Release 标题仍为 `PromptImageManager v2.3.1`。
2. 确认 `draft=false`、`prerelease=false`。
3. 确认两个安装包附件均可见且大小正确。
4. 确认正文包含软件功能与特色介绍。
5. 确认 SHA256 与本地文件一致。

## Release 正文草案

```md
## 软件简介

PromptImageManager 是一款面向 AI 生图工作流的提示词与图片管理工具。它可以帮助你集中整理正向提示词、反向提示词、参考图片、分类标签和版本记录，并支持 PC 与 Android 在同一局域网内同步数据。

## 适合谁使用

- 经常整理 AI 生图提示词的创作者
- 需要按项目、风格、用途管理提示词素材的用户
- 希望在电脑和安卓设备之间同步提示词库的用户
- 想要本地保存数据、不依赖云端账号的用户

## 核心功能

- 提示词库管理：集中保存、编辑、搜索和复用提示词
- 分类与标签：通过文件夹、颜色和标签组织素材
- 图片管理：为提示词绑定参考图、封面图和本地图片资源
- 版本记录：保留提示词修改历史，方便回溯和复用
- 导入导出：支持本地数据备份、恢复和迁移
- PC 与 Android 互通：支持同一局域网内拉取、回传和双向同步

## v2.3.1 版本亮点

- 局域网同步增强：Android 支持从 PC 拉取、回传到 PC 和双向同步
- 动态端口连接：PC 端口占用时可从 `8888` 回退到 `8889-8897`
- 移动端搜索增强：自动扫描常用端口，并优先检测最近连接设备
- 同步安全保护：写入类接口增加配对令牌校验，避免未授权写入
- 图片查看优化：PC 端支持图片缩放、平移、双击复位和缩放比例显示
- 桌面体验优化：侧边栏支持收起，默认窗口调整为更适合桌面使用的 16:9 比例

## 安装包下载

- Android：`PromptImageManager-v2.3.1-Android.apk`

Android 首次安装时，如系统提示拦截，请在系统设置中允许当前浏览器或文件管理器安装未知来源应用。
```
