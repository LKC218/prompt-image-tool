# 交互动效全面优化计划

> 版本：v1.0.0
> 创建日期：2026-07-07
> 范围：PC 端所有可交互元素的鼠标点击动效、鼠标悬浮、图标点击动效全面优化，对标 TDesign 企业级设计体系交互反馈规范

## 一、思考与分析

### 1.1 技能与上下文嗅探

- 项目为原生 ES Module + Vite，无 Vue/React 框架，无法直接引入 TDesign 组件库。
- 方案为提取 TDesign 交互设计规范作为设计 Token，以纯 CSS 变量 + 轻量 JS 工具实现。
- 已确认 PC 端全局按钮微交互基线在 `src/css/pc.css:8388-8545`（`:is()` 选择器集中覆盖 38+ 类可点击元素）。

### 1.2 当前现状

- 全局基线 transition: 0.16s ease，TDesign 标准 0.2s cubic-bezier(.38,0,.24,1)。
- 无 Ripple 波纹点击反馈（TDesign 核心交互特征）。
- 8 处 focus-visible 外轮廓使用 3px 半透明蓝，TDesign 规范 2px solid 品牌色。
- 图标操作按钮 active 仅 scale(0.98)，缺乏背景加深反馈。
- 复制操作无成功视觉反馈动画。
- 导航栏动效已完善（hover 上浮 + 蓝色染色 + 点击回弹），仅同步 timing。

## 二、实施内容

### Phase 1：TDesign 交互 Token 注入

- 在 `.pc-app` 变量区新增 TDesign Token 变量组（`src/css/pc.css:71-85`）：
  - `--td-motion-standard: 0.2s`（统一替代 0.16s/0.18s）
  - `--td-motion-easing: cubic-bezier(.38, 0, .24, 1)`（统一替代 ease）
  - `--td-motion-easing-emphasized: cubic-bezier(.2, .8, .2, 1)`（弹性回弹专用）
  - `--td-bg-color-container-hover: rgba(0,0,0,0.04)`
  - `--td-bg-color-container-active: rgba(0,0,0,0.07)`
  - `--td-brand-color: var(--pc-blue)`
  - `--td-ripple-color: rgba(45, 140, 255, 0.22)`
  - `--td-focus-outline: 2px solid var(--td-brand-color)`

### Phase 2：全局基线升级

- 按钮组 transition: `0.16s ease` → `var(--td-motion-standard) var(--td-motion-easing)`
- 卡片组 transition: `0.18s ease` → `var(--td-motion-standard) var(--td-motion-easing)`
- focus-visible 外轮廓: `3px solid rgba(...)` → `var(--td-focus-outline)` (2px)
- 全局基线添加 `position: relative`（波纹定位基础）

### Phase 3：Ripple 波纹系统

- 新建 `src/js/ripple.js`：
  - 事件委托模式，document 级 pointerdown 监听
  - 通过选择器匹配 40+ 类可点击元素（含导航项）
  - 点击位置创建 `.td-ripple` span，animationend 自动清理
  - prefers-reduced-motion 检测，禁用波纹创建
- `src/js/pc-app.js` 在 `mount()` 中调用 `initRipple(appEl)`
- `src/css/pc.css` 新增 `.td-ripple` 样式 + `@keyframes td-ripple-spread`

### Phase 4：统一 hover/active/focus-visible

- 导航栏 transition 升级到 TDesign 标准（保留弹性缓动用于 transform）
- 8 处 `outline: 3px` 统一为 `var(--td-focus-outline)` (2px)
- 图标按钮新增 active 状态：`background: var(--td-bg-color-container-active); transform: scale(0.94)`
- 详情操作按钮 transition 升级

### Phase 5：复制成功反馈

- `src/css/pc.css` 新增 `.td-copy-success` + `@keyframes td-copy-success-pulse`（弹跳 + 背景变绿 0.5s）
- `src/js/pc-detail.js` 两处复制按钮点击事件添加 `td-copy-success` class 切换

### Phase 6：低动效兜底

- 全局 `prefers-reduced-motion` 块新增 `.td-ripple { display: none }` 和 `.td-copy-success { animation: none }` 兜底
