## 十、T7 — 标签增删动画实现（P2）

### 10.1 设计规范要求

| 动画 | 时长 | 说明 |

| 标签添加 | 0.15s | 缩放弹入 |

### 10.2 实现设计

#### F7.1 标签删除动画

- 删除标签时，先给目标标签添加退出动画类，动画结束后再从 DOM 移除
- CSS 动画：
  ```css
  .m-tag-pill.m-tag-exit {
      animation: m-tag-exit 0.15s ease forwards;
  }
  @keyframes m-tag-exit {
      from { transform: scale(1); opacity: 1; }
      to { transform: scale(0); opacity: 0; }
  }
  ```
- JS 逻辑：
  ```javascript
  const tagEl = pageEl.querySelector(`.m-tag-pill[data-tag="${tag}"]`);
  if (tagEl) {
      tagEl.classList.add('m-tag-exit');
      tagEl.addEventListener('animationend', () => {
          selectedTags = selectedTags.filter(t => t !== tag);
          renderTags(pageEl);
      }, { once: true });
  }
  ```

#### F7.2 标签添加动画

- 标签选择弹窗确认后，新添加的标签添加入场动画类
- CSS 动画：
  ```css
  .m-tag-pill.m-tag-enter {
      animation: m-tag-enter 0.15s ease forwards;
  }
  @keyframes m-tag-enter {
      from { transform: scale(0); opacity: 0; }
      to { transform: scale(1); opacity: 1; }
  }
  ```
- JS 逻辑：渲染标签后，对新增标签添加 `m-tag-enter` 类

## 十五、风险提示

| 风险项 | 影响范围 | 应对措施 |
|--------|---------|---------|
| SQLite ALTER TABLE 兼容性 | 旧版本数据升级 | 在 `init()` 中检测字段是否存在再执行 ALTER，避免重复添加 |
| 标签 JSON 字符串解析失败 | 标签数据读取 | 使用 try-catch 包裹 JSON.parse，失败时回退为空数组 |
| ActionSheet 复用导致样式冲突 | 多个弹窗共用基础样式 | 为每个弹窗使用独立 CSS 类前缀（如 `m-tag-picker`、`m-folder-picker`） |
| 右滑手势与页面滚动冲突 | 横向滑动误触发返回 | 严格限制起始位置在左边缘 20px 内，且水平滑动距离需大于垂直距离 |
| PC 端 API 未同步更新导致双端数据不一致 | PC 端无法读取新字段 | PC 端 Python 后端同步更新，新字段为可选参数，缺失时不影响现有功能 |
| Mock 标签数据硬编码 | 后续扩展需修改代码 | 标签数据集中定义为常量，后续迁移到数据库时仅需修改数据源 |
