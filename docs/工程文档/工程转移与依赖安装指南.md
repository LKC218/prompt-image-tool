### 1.3 打包命令

使用 7-Zip（推荐，压缩率更高）：

```powershell
cd <项目父目录>

7z a -t7z prompt-image-tool-transfer.7z prompt-image-tool\ ^
  -xr!node_modules ^
  -xr!.venv ^
  -xr!src-tauri\target ^
  -xr!src-tauri\gen\schemas ^
  -xr!android\.gradle ^
  -xr!android\app\build ^
  -xr!android\app\src\main\assets\public ^
  -xr!android\app\src\main\assets\capacitor.config.json ^
  -xr!android\app\src\main\assets\capacitor.plugins.json ^
  -xr!android\app\src\main\res\xml\config.xml ^
  -xr!android\local.properties ^
  -xr!build\build ^
  -xr!build\dist ^
  -xr!*.exe ^
  -xr!*.apk ^
  -xr!*.zip
```

使用 Windows 自带 tar：

```powershell
cd <项目父目录>

tar -a -c -f prompt-image-tool-transfer.zip ^
  --exclude=node_modules ^
  --exclude=.venv ^
  --exclude=src-tauri\target ^
  --exclude=src-tauri\gen\schemas ^
  --exclude=android\.gradle ^
  --exclude=android\app\build ^
  --exclude=android\local.properties ^
  --exclude=build\build ^
  --exclude=build\dist ^
  --exclude=*.exe ^
  --exclude=*.apk ^
  --exclude=*.zip ^
  prompt-image-tool
```

> 精简后压缩包预计 **15-20 MB**。

## 二、新电脑：基础软件安装

### 2.1 必装软件（所有开发场景）

#### Node.js

- 下载地址：https://nodejs.org/
- 推荐版本：LTS（长期支持版）
- 安装时勾选"Add to PATH"
- 验证安装：

```powershell
node --version
npm --version
```

#### Python 3.9+

- 下载地址：https://www.python.org/downloads/
- 安装时**必须**勾选：
  - ✅ Add Python to PATH
  - ✅ Install pip
- 验证安装：

```powershell
python --version
pip --version
```

### 2.2 按需安装软件

#### Rust（仅开发 Tauri 桌面端需要）

- 下载地址：https://rustup.rs/
- 最低版本要求：**1.77.2+**
- 安装方式：下载 `rustup-init.exe` 并运行，选择默认安装
- 验证安装：

```powershell
rustc --version
cargo --version
```

> **注意**：Rust 安装后会自动配置环境变量，需重启终端生效。

#### JDK 17+（仅开发 Android 端需要）

- 推荐版本：JDK 17（与项目兼容性最佳）
- 下载地址：https://adoptium.net/
- 安装后设置环境变量：

```powershell
# 系统环境变量
JAVA_HOME = C:\Program Files\Eclipse Adoptium\jdk-17.x.x-hotspot
Path 中添加 %JAVA_HOME%\bin
```

- 验证安装：

```powershell
java -version
javac -version
```

#### Android SDK（仅开发 Android 端需要）

- 安装方式一：安装 Android Studio（推荐，自带 SDK Manager）
  - 下载地址：https://developer.android.com/studio
- 安装方式二：仅安装命令行工具
  - 下载地址：https://developer.android.com/studio#command-line-tools-only

- 通过 SDK Manager 安装以下组件：
  - Android SDK Platform 36（compileSdkVersion = 36）
  - Android SDK Build-Tools
  - Android SDK Platform-Tools
  - Android SDK Command-line Tools

- 设置环境变量：

```powershell
# 系统环境变量
ANDROID_HOME = C:\Users\你的用户名\AppData\Local\Android\Sdk
Path 中添加：
  %ANDROID_HOME%\platform-tools
  %ANDROID_HOME%\build-tools\最新版本
```

- 验证安装：

```powershell
adb version
```

#### NSIS（仅打包 PC 安装包需要）

- 下载地址：https://nsis.sourceforge.io/Download
- 安装后确保 `makensis` 在 PATH 中
- 验证安装：

```powershell
makensis /version
```

### 3.2 还原前端依赖（npm）

```powershell
cd D:\Projects\prompt-image-tool

# 根据 package-lock.json 精确还原所有依赖

```

> **为什么用 `npm ci` 而不是 `npm install`？**
> - `npm ci` 严格按照 `package-lock.json` 安装，保证版本一致
> - `npm install` 可能升级依赖，导致版本漂移
> - 如果 `package-lock.json` 丢失，只能用 `npm install`，但版本可能不一致

## 六、关键文件安全提醒

| 文件 | 重要性 | 说明 |
|---|---|---|
| `android/prompt-image-manager.keystore` | 🔴 极高 | Android 签名密钥，丢失后无法更新已发布的 APK |
| `android/keystore.properties` | 🔴 极高 | 含签名密钥密码，需安全保管 |
| `package-lock.json` | 🟡 高 | npm 精确版本锁定，丢失可能导致依赖版本不一致 |
| `Cargo.lock` | 🟡 高 | Rust 精确版本锁定，丢失可能导致编译结果不同 |
| `gradle-wrapper.jar` | 🟡 高 | Gradle Wrapper 自身，丢失需手动安装 Gradle |
| `build/app.spec` | 🟡 高 | PyInstaller 打包配置，含隐式导入声明 |
| `build/installer.nsi` | 🟢 中 | NSIS 安装脚本，可重建但较繁琐 |
