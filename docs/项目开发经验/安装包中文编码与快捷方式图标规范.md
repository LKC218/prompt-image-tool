# 安装包中文编码与快捷方式图标规范

## 适用范围

适用于 Windows 安装包、批处理构建入口、Python 构建脚本和发布产物校验。

## 编码规范

- 构建脚本和 NSIS 脚本统一保存为 UTF-8。
- Python 构建脚本启动后显式设置标准输出编码：

```python
if hasattr(sys.stdout, "reconfigure"):
    sys.stdout.reconfigure(encoding="utf-8")
if hasattr(sys.stderr, "reconfigure"):
    sys.stderr.reconfigure(encoding="utf-8")
```

- NSIS 脚本必须启用 Unicode：

```nsi
Unicode true
!insertmacro MUI_LANGUAGE "SimpChinese"
```

- NSIS 构建命令必须指定 UTF-8 输入：

```powershell
makensis /INPUTCHARSET UTF8 installer.nsi
```

- PowerShell 查看中文文件时优先使用：

```powershell
Get-Content -Raw -Encoding UTF8 <path>
```

## 快捷方式图标规范

- 安装器图标和卸载器图标统一使用 `build/icon.ico`。
- 安装时必须将图标复制到安装目录。
- 桌面快捷方式、开始菜单快捷方式和 Windows 卸载项必须统一指向安装目录内的图标文件。

```nsi
Icon "icon.ico"
UninstallIcon "icon.ico"
File "icon.ico"
CreateShortCut "$DESKTOP\生图提示词管理器.lnk" "$INSTDIR\PromptImageManager.exe" "" "$INSTDIR\icon.ico" 0
WriteRegStr HKCU "Software\Microsoft\Windows\CurrentVersion\Uninstall\PromptImageManager" "DisplayIcon" "$INSTDIR\icon.ico"
```

## 脚本预检建议

正式构建脚本应在执行 PyInstaller 和 NSIS 前检查：

- `build/icon.ico` 存在。
- `build/installer.nsi` 包含 `Unicode true`。
- `build/installer.nsi` 包含简体中文语言宏。
- 桌面快捷方式和开始菜单快捷方式均绑定 `$INSTDIR\icon.ico`。
- 卸载项 `DisplayIcon` 指向 `$INSTDIR\icon.ico`。

这样可以在构建早期发现配置缺失，避免安装包生成后才暴露中文乱码或默认白板图标问题。
