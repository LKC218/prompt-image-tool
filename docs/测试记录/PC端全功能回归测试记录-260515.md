1. Tauri 桌面壳测试存在一个 Rust 未使用导入警告。
   - 处理：移除 `src-tauri/src/lib.rs` 测试模块中的冗余 `use super::*;`。
   - 复测：`npm.cmd run test:rust` 通过且不再输出该警告。

2. 首轮浏览器 CRUD 冒烟中捕获到一次旧提示词 ID 的 404 控制台噪声。
   - 判断：该 404 来自浏览器本地旧草稿或旧会话状态，不属于当前保存链路缺陷。
   - 证据：清空 `localStorage` 后隔离复测，创建、二次更新、列表刷新、删除均返回 200，控制台无错误。
