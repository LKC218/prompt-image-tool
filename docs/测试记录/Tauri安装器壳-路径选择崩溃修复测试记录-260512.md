# Tauri安装器壳-路径选择崩溃修复测试记录

## 测试范围

- 安装位置页点击“浏览”不再通过 PowerShell/WinForms 弹目录窗口。
- 目录选择取消、选择成功、路径校验和下一步跳转链路。
- 安装器壳 release 构建与真实窗口交互。
- 选择路径后的真实安装落点验证。

## 修复摘要

- 前端浏览入口已改为通过 `window.__TAURI__.dialog.open` 异步打开目录选择窗口，避免 Rust 命令同步阻塞。
- 后端同步 `choose_install_dir()` 已移除，避免误用同步目录对话框导致前端停留“选择中”。
- 前端增加浏览中的工作态，避免重复点击“浏览”造成多重弹窗或状态覆盖。
- 选择成功后继续通过 `validate_install_path` 校验真实路径，取消选择时保留当前路径。
