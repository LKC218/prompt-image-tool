# 界面3：提示词详情

> 版本：1.0 | 最后更新：2026-05-05

---

## 1. 页面定位与目标

详情页用于查看单条提示词的完整内容，包括：
- 查看封面大图
- 查看正向/负向提示词并复制
- 查看风格模型、比例、标签等元信息
- 查看版本记录
- 执行编辑/复制/更多操作

**核心目标**：让用户快速获取提示词内容并复制使用。

## 3. 视觉规范

### 3.1 页面专用颜色

| 元素 | 颜色 | 说明 |
|------|------|------|
| 封面图区域 | 圆角 28px | `--m-radius-xl` |
| 正向提示词卡片背景 | `--m-green-light` | 浅绿色 |
| 正向提示词卡片标题 | `#3D9942` | 深绿色 |
| 正向提示词复制按钮 | `--m-green` 背景 + 白色字 | 绿色按钮 |
| 负向提示词卡片背景 | `--m-pink-bg` | 浅粉色 |
| 负向提示词卡片标题 | `#D4567F` | 深粉色 |
| 负向提示词复制按钮 | `--m-pink` 背景 + 白色字 | 粉色按钮 |
| 信息卡片 | `--m-surface` | 白色 |
| 编辑按钮 | `--m-yellow` 背景 + 白色字 | 黄色 |
| 复制按钮 | `--m-blue` 背景 + 白色字 | 蓝色 |
| 更多按钮 | `--m-surface2` 背景 + `--m-text2` 字 | 灰色 |

### 3.2 尺寸规范

| 属性 | 值 |
|------|-----|
| 封面图高度 | 200px，宽度 100%，圆角 28px |
| 正/负向卡片圆角 | `--m-radius-lg`（24px） |
| 正/负向卡片内边距 | 16px |
| 提示词文字字号 | `--m-font-body`（16px） |
| 提示词文字行高 | 1.8 |
| 底部操作栏高度 | 64px + 安全区 34px |

## 5. 数据绑定

| 数据 | 来源 | API/方法 |
|------|------|---------|
| 封面图 | 集合首个版本的首张图片 | `getPromptSet(id)` → `versions[0].images[0]` |
| 标题 | 集合名称 | `promptSet.name` |
| 标签 | 集合标签 | Phase 1 mock |
| 收藏数 | 收藏计数 | Phase 1 mock |
| 正向提示词 | 当前版本 | `version.positive_prompt` |
| 负向提示词 | 当前版本 | `version.negative_prompt` |
| 风格/模型 | 版本元信息 | Phase 1 mock |
| 比例 | 版本元信息 | Phase 1 mock |
| 创建时间 | 集合创建时间 | `promptSet.created_at` |
| 版本列表 | 集合所有版本 | `promptSet.versions` |

## 6. 交互规范

### 6.1 交互状态矩阵

| 元素 | 默认态 | 触摸态 | 选中态 |
|------|--------|--------|--------|
| 返回按钮 | 箭头图标 | `--m-text` 加深 | — |
| 收藏星标 | 灰色轮廓/黄色填充 | `scale(1.2)` | 切换 |
| 封面图 | 正常显示 | 轻微缩放 | — |
| 复制按钮 | 彩色背景 | `scale(0.95)` | — |
| 信息卡片 | 白色背景 | — | — |
| 版本记录项 | 白色背景 | 背景→`--m-surface2` | — |
| 底部操作按钮 | 各自颜色 | `scale(0.95)` | — |

### 6.2 手势操作

| 手势 | 目标 | 行为 |
|------|------|------|
| 点击 | 返回按钮 | 返回提示词库页 |
| 点击 | 收藏星标 | 切换收藏状态 |
| 点击 | 封面图 | 全屏查看图片 |
| 点击 | 正向复制按钮 | 复制正向提示词到剪贴板 |
| 点击 | 负向复制按钮 | 复制负向提示词到剪贴板 |
| 点击 | 版本记录项 | 展开版本详情或跳转 |
| 点击 | 编辑按钮 | 跳转到编辑页 |
| 点击 | 复制按钮 | 复制完整提示词（正+负） |
| 点击 | 更多按钮 | 弹出底部动作表 |
| 右滑（边缘） | 页面 | 返回提示词库页 |

### 6.3 动画过渡

| 动画 | 时长 | 说明 |
|------|------|------|
| 页面入场 | 0.25s | 从右侧滑入 |
| 页面退出 | 0.25s | 向右侧滑出 |
| 封面图加载 | 0.25s | 淡入 |
| 复制成功反馈 | 0.3s | 按钮文字变为「已复制✓」，0.3s 后恢复 |
| 收藏切换 | 0.15s | 星标缩放弹跳 |

### 6.4 异常状态

| 状态 | 视觉 | 文案 |
|------|------|------|
| 无封面图 | 显示占位图（可爱插画风格） | — |
| 正向提示词为空 | 卡片显示灰色文字 | 「暂无正向提示词」 |
| 负向提示词为空 | 卡片显示灰色文字 | 「暂无负向提示词」 |
| 无版本记录 | 版本区域显示空状态 | 「暂无版本记录」 |
| 图片加载失败 | 显示占位图 | — |

## 7. 功能映射

| 本页面功能 | 对应现有功能 | 迁移说明 |
|-----------|-------------|---------|
| 封面大图 | 详情页图片网格首张 | 提升为封面展示 |
| 标题+标签 | 详情页名称输入+版本标签 | 改为只读展示 |
| 正向提示词卡片 | 详情页正向提示词Section | 改为浅绿色卡片+复制按钮 |
| 负向提示词卡片 | 详情页负向提示词Section | 改为浅粉色卡片+复制按钮 |
| 信息卡片 | 详情页版本元信息 | 整合为一张卡片 |
| 版本记录 | 详情页版本标签栏 | 改为列表形式 |
| 编辑按钮 | 详情页编辑能力 | 独立按钮，跳转编辑页 |
| 复制按钮 | 详情页复制按钮+预览复制 | 合并为底部操作 |
| 收藏星标 | 无（新增） | Phase 1 仅 UI |
