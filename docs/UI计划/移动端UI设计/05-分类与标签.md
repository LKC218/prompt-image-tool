# 界面5：分类与标签

> 版本：1.1 | 最后更新：2026-05-10

---

## 1. 页面定位与目标

用于管理分类和标签，包括：
- 查看和切换分类管理/标签管理
- 新建分类
- 排序、批量编辑、合并分类、删除
- 拖拽调整分类顺序

**核心目标**：让用户高效组织提示词的分类体系。

## 3. 视觉规范

### 3.1 页面专用颜色

| 元素 | 颜色 | 说明 |
|------|------|------|
| 分段控制器-选中 | 暖白凸起 + `--m-accent` 文字 | 内凹胶囊槽中的激活页签 |
| 分段控制器-未选中 | 内凹胶囊槽 + `--m-text2` 文字 | 灰色 |
| 新建分类按钮 | 暖白凸起 + `--m-accent` 文字 | 保留加号作为主操作识别 |
| 分类列表卡片 | `--m-neu-bg` 凸起面板 | 白色同材质拟态卡片 |
| 分类图标底色 | 各分类对应色 | 彩色圆角方形 |
| 快速操作按钮 | 暖白凸起 + 语义色图标 | 操作语义由图标色表达，按压内凹 |
| 底部提示横幅 | 暖白凸起 + 兔子插画 | 低对比提示面板 |

### 3.2 分类图标颜色映射

| 分类 | 图标底色 | 图标颜色 |
|------|---------|---------|
| 插画风格 | `--m-blue-light` | `--m-blue` |
| 写实摄影 | `--m-yellow-light` | `#C4A030` |
| 科幻未来 | `--m-purple-light` | `#8B6FCC` |
| 国风古韵 | `--m-pink-light` | `#D4567F` |
| 场景设计 | `--m-green-light` | `#3D9942` |
| 人物角色 | `#FFE0CC` | `#E07020` |

分类页面通过 `.m-category-page` 建立局部拟态令牌；顶部返回、分段控制器、列表、快捷操作、标签胶囊及创建/确认弹窗按凸起、内凹和按压层级统一。分类和标签的语义色保留在图标与标签本身，避免在大面积背景中重复使用纯色。

## 5. 数据绑定

| 数据 | 来源 | API/方法 |
|------|------|---------|
| 分类列表 | 文件夹查询 | `getFolders()` |
| 各分类下提示词数 | 聚合查询 | 按 folder_id 分组计数 |
| 标签列表 | 前端聚合 | `aggregateTags(promptSets)`，包含自定义标签与使用次数 |
| 标签颜色 | 共享语义映射 | 仅预设语义使用专属色，其他标签保持默认浅色样式 |

## 6. 交互规范

### 6.1 交互状态矩阵

| 元素 | 默认态 | 触摸态 | 选中态 |
|------|--------|--------|--------|
| 分段控制器-选中 | 暖白凸起+主题色文字 | 内凹 | 凸起页签 |
| 分段控制器-未选中 | 灰色文字+内凹槽 | 内凹 | — |
| 分类列表项 | 暖白凸起 | 内凹并下移 1px | — |
| 三点菜单 | 灰色图标 | 图标→`--m-text` | 弹出菜单 |
| 拖拽手柄 | 灰色点阵 | 图标→`--m-text2` | 可拖拽 |
| 快速操作按钮 | 暖白凸起+语义色图标 | 内凹并下移 1px | — |
| 兔子横幅 | 暖白凸起 | — | — |

### 6.2 手势操作

| 手势 | 目标 | 行为 |
|------|------|------|
| 点击 | 分段控制器 | 切换分类/标签管理视图 |
| 点击 | 新建分类 | 弹出新建分类弹窗（名称+颜色） |
| 点击 | 分类列表项 | 展开该分类下的提示词列表 |
| 点击 | 三点菜单 | 弹出操作菜单（重命名/改颜色/删除） |
| 长按+拖拽 | 分类列表项 | 调整分类顺序 |
| 点击 | 排序按钮 | 进入排序模式 |
| 点击 | 批量编辑 | 进入多选编辑模式 |
| 点击 | 合并分类 | 弹出合并选择弹窗 |
| 点击 | 删除按钮 | 弹出删除确认弹窗 |
| 点击 | 返回按钮 | 返回上一页 |
| 点击 | 标签胶囊 | 打开该标签操作菜单 |
| 点击 | 标签筛选 | 保持既有中性新拟态高亮当前筛选条件 |

### 6.3 动画过渡

| 动画 | 时长 | 说明 |
|------|------|------|
| 分段切换 | 0.25s | 内容区淡入切换 |
| 列表项拖拽 | 实时 | 跟随手指移动，其他项自动让位 |
| 列表项删除 | 0.25s | 向左滑出+高度收缩 |
| 快速操作按钮 | 0.15s | `scale(0.95)` 触摸反馈 |

### 6.4 异常状态

| 状态 | 视觉 | 文案 |
|------|------|------|
| 无分类 | 空状态插画 | 「还没有分类，点击上方新建吧~」 |
| 删除确认 | 弹出确认弹窗 | 「确定删除该分类？分类下的提示词将变为未分类」 |
| 合并冲突 | 弹出选择弹窗 | 「选择要合并的目标分类」 |
