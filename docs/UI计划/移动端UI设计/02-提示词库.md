# 界面2：提示词库

> 版本：1.0 | 最后更新：2026-05-05

---

## 1. 页面定位与目标

提示词库是提示词列表页，用于：
- 搜索、筛选提示词
- 浏览所有提示词卡片
- 收藏/取消收藏
- 快速操作（重命名/复制/删除/移动）

**核心目标**：让用户高效找到目标提示词并快速操作。

## 3. 视觉规范

### 3.1 页面专用颜色

| 元素 | 颜色 | 说明 |
|------|------|------|
| 页面标题 | `--m-text` | 黑色大标题 |
| 筛选标签-选中 | `--m-blue` 描边 + `--m-blue-light` 背景 | 蓝色选中态 |
| 筛选标签-默认 | `--m-border` 描边 + 透明背景 | 灰色默认态 |
| 列表卡片 | `--m-surface` | 白色 |
| 收藏星标-已收藏 | `--m-yellow` 填充 | 黄色实心星 |
| 收藏星标-未收藏 | `--m-text3` 轮廓 | 灰色空心星 |

### 3.2 列表卡片尺寸

| 属性 | 值 |
|------|-----|
| 卡片圆角 | `--m-radius-lg`（24px） |
| 缩略图尺寸 | 80×80px，圆角 12px |
| 卡片内边距 | 12px |
| 标题字号 | `--m-font-h3`（20px） |
| 描述字号 | `--m-font-caption`（14px），最多2行 |
| 时间字号 | `--m-font-micro`（12px） |

## 5. 数据绑定

| 数据 | 来源 | API/方法 |
|------|------|---------|
| 提示词列表 | 全量查询 | `getPromptSets()` |
| 搜索结果 | 关键词过滤 | `filterPromptSets(keyword)` |
| 筛选结果 | 分类/收藏/标签过滤 | 客户端筛选 |
| 收藏状态 | prompt_set.is_favorite | Phase 1 mock |

### 筛选逻辑

| 筛选项 | 筛选逻辑 |
|--------|---------|
| 全部 | 无筛选，显示所有集合 |
| 收藏 | `is_favorite === true` |
| 最近使用 | 按更新时间排序 |
| 场景 | 标签包含「场景」 |
| 人物 | 标签包含「人物」 |

## 6. 交互规范

### 6.1 交互状态矩阵

| 元素 | 默认态 | 触摸态 | 选中态 |
|------|--------|--------|--------|
| 筛选标签 | 灰色描边+透明底 | — | 蓝色描边+浅蓝底+蓝色字 |
| 列表卡片 | 白色背景 | 背景→`--m-surface2` | — |
| 收藏星标 | 灰色轮廓/黄色填充 | `scale(1.2)` | 切换收藏状态 |
| 三点菜单 | 灰色图标 | 图标→`--m-text` | 弹出菜单 |

### 6.2 手势操作

| 手势 | 目标 | 行为 |
|------|------|------|
| 点击 | 筛选标签 | 切换筛选条件，刷新列表 |
| 点击 | 列表卡片 | 跳转到提示词详情页 |
| 点击 | 收藏星标 | 切换收藏状态（Phase 1 仅 UI） |
| 点击 | 三点菜单 | 弹出底部动作表（重命名/复制/移动/删除） |
| 横向滑动 | 筛选标签栏 | 滚动查看更多筛选项 |
| 下拉 | 列表 | 刷新数据 |

### 6.3 动画过渡

| 动画 | 时长 | 说明 |
|------|------|------|
| 筛选切换 | 0.15s | 列表淡入刷新 |
| 收藏切换 | 0.15s | 星标缩放弹跳 |
| 列表项入场 | 0.25s | 依次淡入+上移 |

### 6.4 异常状态

| 状态 | 视觉 | 文案 |
|------|------|------|
| 无提示词 | 空状态插画+文字 | 「还没有提示词，点击下方加号创建吧~」 |
| 搜索无结果 | 空状态插画+文字 | 「未找到匹配的提示词」 |
| 筛选无结果 | 空状态插画+文字 | 「该分类下暂无提示词」 |
| 加载中 | 骨架屏 | — |
