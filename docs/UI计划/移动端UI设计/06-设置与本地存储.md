# 界面6：设置与本地存储

> 版本：1.0 | 最后更新：2026-05-05

---

## 1. 页面定位与目标

展示应用设置和本地存储信息，强调本地安全：
- 外观设置（深色模式/主题色/跟随系统）
- 本地存储数据查看
- 数据备份与恢复
- 本地安全提示

**核心目标**：让用户安心使用，明确数据安全，提供数据管理入口。

## 3. 视觉规范

### 3.1 页面专用颜色

| 元素 | 颜色 | 说明 |
|------|------|------|
| 柯基安全横幅背景 | 浅蓝色渐变 | `linear-gradient(135deg, #E8F4FF, #D0E8FF)` |
| 对话气泡 | `--m-surface` 白色 | 白色圆角气泡 |
| 设置卡片 | `--m-surface` 白色 | 白色 |

## 6. 交互规范

### 6.1 交互状态矩阵

| 元素 | 默认态 | 触摸态 | 选中态 |
|------|--------|--------|--------|
| 深色模式开关 | 灰色轨道 | — | 绿色轨道+白色滑块 |
| 跟随系统开关 | 灰色轨道 | — | 绿色轨道+白色滑块 |
| 主题色圆点 | 对应颜色 | `scale(1.15)` | 带白色勾选+外阴影 |
| 设置行 | 白色背景 | 背景→`--m-surface2` | — |
| 备份恢复卡片 | 各自浅色背景 | `scale(0.97)` | — |
| 安全提示栏 | 浅绿背景 | 背景→加深 | — |

### 6.2 手势操作

| 手势 | 目标 | 行为 |
|------|------|------|
| 点击 | 深色模式开关 | 切换暗色/亮色主题 |
| 点击 | 主题色圆点 | 选中该主题色 |
| 点击 | 跟随系统开关 | 切换跟随系统偏好 |
| 点击 | 本地备份 | 触发数据导出 |
| 点击 | 导入数据 | 触发文件选择器 |
| 点击 | 导出 JSON | 导出为 .json 文件 |
| 点击 | 从 JSON 导入 | 选择 .json 文件导入 |
| 点击 | 安全提示栏 | 展开安全说明详情 |

### 6.3 动画过渡

| 动画 | 时长 | 说明 |
|------|------|------|
| 主题切换 | 0.25s | 背景色+文字色渐变 |
| 开关切换 | 0.2s | 滑块滑动 |
| 主题色选中 | 0.15s | 勾选标记缩放弹入 |
| 备份恢复卡片触摸 | 0.1s | `scale(0.97)` |

### 6.4 异常状态

| 状态 | 视觉 | 文案 |
|------|------|------|
| 导出失败 | Toast 错误提示 | 「导出失败，请重试」 |
| 导入失败 | Toast 错误提示 | 「导入失败，文件格式不正确」 |
| 存储空间不足 | 安全提示栏变红 | 「存储空间不足，请清理数据」 |
| 数据加载中 | 骨架屏 | — |
