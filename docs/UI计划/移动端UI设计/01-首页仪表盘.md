# 界面1：首页仪表盘

> 版本：1.0 | 最后更新：2026-05-05

---

## 1. 页面定位与目标

首页是用户进入 App 后的仪表盘，用于：
- 快速了解提示词管理全貌（统计数据）
- 查看最近使用的提示词
- 浏览收藏分类
- 快速创建新提示词或导入提示词

**核心目标**：让用户一眼掌握全局，一键进入操作。

## 3. 视觉规范

### 3.1 页面专用颜色

| 元素 | 颜色 | 说明 |
|------|------|------|
| 欢迎区背景 | `--m-bg` | 奶油白 |
| 搜索栏背景 | `--m-surface` | 白色 |
| 统计卡片-提示词总数 | `--m-blue-light` + `--m-blue` | 浅蓝底+蓝字 |
| 统计卡片-分类 | `--m-yellow-light` + `#C4A030` | 浅黄底+深黄字 |
| 统计卡片-标签 | `--m-purple-light` + `#8B6FCC` | 浅紫底+深紫字 |
| 统计卡片-收藏 | `--m-pink-light` + `#D4567F` | 浅粉底+深粉字 |
| 最近使用卡片 | `--m-surface` | 白色 |
| 收藏分类卡片 | 各分类对应色 | 彩色背景 |
| 快速创建卡片 | `--m-surface` | 白色 |

### 3.2 间距

| 区域 | 间距 |
|------|------|
| 页面水平内边距 | `--m-space-lg`（16px） |
| 模块间间距 | `--m-space-xl`（20px） |
| 统计卡片间距 | `--m-space-sm`（8px） |
| 最近使用列表项间距 | `--m-space-sm`（8px） |

## 6. 交互规范

### 6.1 交互状态矩阵

| 元素 | 默认态 | 触摸态 | 选中态 | 禁用态 |
|------|--------|--------|--------|--------|
| 搜索栏 | 白色背景，灰色占位文字 | 边框→`--m-blue` | — | — |
| 统计卡片 | 浅色背景 | `scale(0.97)` + 阴影减弱 | — | — |
| 最近使用项 | 白色背景 | 背景→`--m-surface2` | — | — |
| 收藏星标 | 灰色轮廓 | — | `--m-yellow` 填充 | — |
| 三点菜单 | 灰色图标 | 图标→`--m-text` | — | — |
| 收藏分类卡片 | 彩色背景 | `scale(0.97)` | — | — |
| 快速创建卡片 | 白色背景 | 边框→`--m-pink`，背景→`--m-pink-bg` | — | — |

### 6.2 手势操作

| 手势 | 目标 | 行为 |
|------|------|------|
| 点击 | 搜索栏 | 聚焦输入框，弹出键盘 |
| 点击 | 最近使用项 | 跳转到提示词详情页 |
| 点击 | 收藏星标 | 切换收藏状态 |
| 点击 | 三点菜单 | 弹出操作菜单（重命名/复制/删除/移动） |
| 点击 | 收藏分类卡片 | 跳转到提示词库，筛选该分类 |
| 点击 | 快速创建-新建 | 跳转到新建提示词页 |
| 点击 | 快速创建-导入 | 触发文件选择器 |
| 点击 | 查看全部 | 跳转到提示词库页 |
| 下拉 | 页面 | 刷新数据 |

### 6.3 动画过渡

| 动画 | 时长 | 缓动 | 说明 |
|------|------|------|------|
| 页面入场 | 0.25s | ease | 淡入 |
| 统计卡片入场 | 0.25s | ease | 依次淡入+上移，间隔 50ms |
| 最近使用列表入场 | 0.25s | ease | 依次淡入+上移，间隔 30ms |
| 收藏星标切换 | 0.15s | ease | 缩放弹跳 `scale(1→1.3→1)` |
| 卡片触摸反馈 | 0.1s | ease | `scale(0.97)` |

### 6.4 异常状态

| 状态 | 视觉 | 文案 |
|------|------|------|
| 无提示词 | 统计卡片数字显示 0，最近使用区域显示空状态插画 | 「还没有提示词，点击下方快速创建吧~」 |
| 搜索无结果 | 搜索栏下方显示空状态 | 「未找到匹配的提示词」 |
| 数据加载中 | 统计卡片显示骨架屏 | — |
| 网络错误 | 不适用（本地应用） | — |
