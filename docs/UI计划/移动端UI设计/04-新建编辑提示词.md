# 界面4：新建/编辑提示词

> 版本：1.0 | 最后更新：2026-05-05

---

## 1. 页面定位与目标

用于新建或编辑提示词，包含：
- 输入标题、正向/负向提示词
- 管理标签
- 选择分类、比例、风格预设
- 更多选项（折叠）

**核心目标**：提供清晰高效的表单填写体验，减少输入负担。

## 5. 数据绑定

| 数据 | 来源 | 说明 |
|------|------|------|
| 标题 | `promptSet.name` | 新建时为空 |
| 正向提示词 | `version.positive_prompt` | 新建时为空 |
| 负向提示词 | `version.negative_prompt` | 新建时为空 |
| 标签 | `promptSet.tags` | Phase 1 mock |
| 分类 | `promptSet.folder_id` | 映射为文件夹名称 |
| 比例 | `version.aspect_ratio` | Phase 1 mock，默认 1:1 |
| 风格预设 | `version.style_preset` | Phase 1 mock |

### 5.1 保存逻辑

```
点击「保存」
├── 校验：标题不为空
├── 校验：正向提示词不为空
├── 新建模式：createPromptSet() → createVersion()
├── 编辑模式：updatePromptSetName() → updatePromptSet()
└── Toast: 「已保存」→ 返回详情页
```

## 6. 交互规范

### 6.1 交互状态矩阵

| 元素 | 默认态 | 焦点态 | 触摸态 |
|------|--------|--------|--------|
| 输入框 | `--m-border` 边框 | `--m-pink` 边框 + 粉色外阴影 | — |
| 清空按钮 | 绿色文字 | — | 文字加深 |
| 标签 × 按钮 | 浅紫背景 | — | 背景→深紫，文字→白 |
| 添加标签 | 蓝色文字 | — | 文字加深 |
| 分类卡片 | 白色+箭头 | — | 背景→`--m-surface2` |
| 比例按钮-默认 | 灰色描边 | — | 描边→`--m-text2` |
| 比例按钮-选中 | 粉色描边+浅粉底 | — | — |
| 风格预设卡片 | 白色+箭头 | — | 背景→`--m-surface2` |
| 更多选项 | 白色+向下箭头 | — | 背景→`--m-surface2` |
| 保存按钮 | 粉色填充 | — | `scale(0.95)` |

### 6.2 手势操作

| 手势 | 目标 | 行为 |
|------|------|------|
| 点击 | 输入框 | 聚焦，弹出键盘 |
| 点击 | 清空按钮 | 清空对应输入框内容 |
| 点击 | 标签 × | 删除该标签 |
| 点击 | 添加标签 | 弹出标签选择弹窗 |
| 点击 | 分类卡片 | 弹出分类选择弹窗 |
| 点击 | 比例按钮 | 选中该比例，取消其他 |
| 点击 | 风格预设卡片 | 弹出风格预设选择弹窗 |
| 点击 | 更多选项 | 展开/收起更多选项 |
| 点击 | 保存按钮 | 校验+保存+返回 |
| 右滑（边缘） | 页面 | 弹出确认「放弃编辑？」 |

### 6.3 动画过渡

| 动画 | 时长 | 说明 |
|------|------|------|
| 页面入场 | 0.25s | 从右侧滑入 |
| 标签删除 | 0.15s | 缩小消失 |
| 标签添加 | 0.15s | 缩放弹入 |
| 更多选项展开 | 0.25s | 高度动画+淡入 |
| 保存按钮反馈 | 0.15s | `scale(0.95)` |

### 6.4 异常状态

| 状态 | 视觉 | 文案 |
|------|------|------|
| 标题为空时点保存 | 标题输入框红色边框 | 「请输入标题」 |
| 正向提示词为空时点保存 | 输入框红色边框 | 「请输入正向提示词」 |
| 返回时有未保存修改 | 弹出确认弹窗 | 「有未保存的修改，确定放弃吗？」 |
| 字数超限 | 字数统计变红 | — |

## 7. 功能映射

| 本页面功能 | 对应现有功能 | 迁移说明 |

| 比例选择 | 无（新增） | Phase 1 仅 UI |
| 参考图片预览 | `version.images` + `storage.getImageUrl()` | 新导入图片即时预览，已保存图片回填可访问地址 |
| 风格预设选择 | 无（新增） | Phase 1 仅 UI |
| 保存按钮 | 自动保存 | 改为显式保存按钮 |
| 清空按钮 | 无（新增） | 快捷操作 |
