# 界面2：提示词库

> 版本：1.0 | 最后更新：2026-05-05

---

## 1. 页面定位与目标

提示词库是提示词列表页，用于：
- 搜索、筛选提示词
- 查看所有提示词卡片
- 快速操作（重命名/复制/删除/移动）

**核心目标**：让用户高效找到目标提示词并快速操作。

## 3. 视觉规范

### 3.1 页面专用颜色

| 元素 | 颜色 | 说明 |
|------|------|------|
| 列表项背景 | `--surface` | 表面色 |
| 文件夹筛选-选中 | `--accent` 背景 + 白色字 | 紫色 |
| 文件夹筛选-默认 | `--surface2` 背景 + `--text2` 文字 | 灰色 |
| 提示词项 | `--surface` 背景 | 表面色 |
| 提示词项 Hover | `--surface2` 背景 | 表面色2 |
| 提示词项选中 | `--accent-bg` 背景 + `--accent` 边框 | 紫色高亮 |

### 3.2 尺寸规范

| 属性 | 值 |
|------|-----|
| 提示词项圆角 | `--radius-md` (12px) |
| 提示词项内边距 | 12px 16px |
| 缩略图尺寸 | 60×60px, 圆角 8px |
| 列表项间距 | 8px |

## 5. 数据绑定

| 数据 | 来源 | API/方法 |
|------|------|---------|
| 提示词列表 | 全量查询 | `getPromptSets()` |
| 搜索结果 | 关键词过滤 | `filterPromptSets(keyword)` |
| 筛选结果 | 分类过滤 | 按 folder_id 筛选 |

### 筛选逻辑

| 筛选项 | 筛选逻辑 |
|--------|---------|
| 全部 | 无筛选，显示所有集合 |
| 未分类 | `folder_id === null` |
| 各分类 | `folder_id === [folder_id]` |

## 6. 交互规范

### 6.1 交互状态矩阵

| 元素 | 默认态 | Hover 态 | 选中态 |
|------|--------|---------|--------|
| 文件夹筛选 | 灰色背景+灰色文字 | 背景→`--surface2` | `--accent` 背景+白色文字 |
| 提示词列表项 | 透明背景 | 背景→`--surface2` | 背景→`--accent-bg` + 边框→`--accent` |
| 三点菜单 | 灰色图标 | 图标→`--text` | — |
| 右键菜单项 | 白色背景 | 背景→`--surface2` | — |

### 6.2 操作流程

```
右键提示词集合 → 弹出右键菜单
           ↓
      选择操作（重命名/复制/移动/删除）
           ↓
      执行操作 → Toast 反馈
```

### 6.3 动画过渡

| 动画 | 时长 | 说明 |
|------|------|------|
| 筛选切换 | 0.25s | 列表淡入刷新 |
| 列表项 Hover | 0.15s | 背景色渐变 |
| 右键菜单显隐 | 0.15s | 淡入淡出+缩放 |
