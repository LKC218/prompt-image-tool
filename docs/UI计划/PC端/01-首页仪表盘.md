# 界面1：首页仪表盘

> 版本：1.2 | 最后更新：2026-05-25

---

## 1. 页面定位与目标

首页是 PC 端进入应用后的仪表盘，用于：

- 快速了解提示词、分类、标签和收藏数量。
- 查看最近使用的提示词，并直接进入详情、收藏或打开更多操作。
- 查看常用分类，并跳转到提示词库分类筛选结果。
- 快速新建提示词或从 JSON 文件导入提示词。

**核心目标**：严格贴近参考图的桌面首屏排版，让创作者一眼掌握全局，并能快速进入下一步操作。

## 5. 数据绑定

| 数据 | 来源 | 方法 |
|------|------|------|
| 提示词总数 | 存储适配层 | `getPromptSets().length` |
| 分类数 | 存储适配层 | `getFolders().length` |
| 标签数 | 前端聚合 | `aggregateTags(promptSets).length` |
| 收藏数 | 前端聚合 | `promptSets.filter(p => p.isFavorite).length` |
| 最近使用 | 提示词列表 | 按 `updatedAt` 倒序取前 6 条 |
| 最近使用缩略图 | 提示词摘要 | `firstImage` + `storage.getImageUrl()` |
| 收藏分类 | 分类列表 | `getFolders()` + 提示词数量聚合 |

## 6. 交互规范

| 元素 | 行为 |
|------|------|
| 搜索输入 | 输入时同步筛选最近使用和收藏分类 |
| 搜索回车 | 带搜索词跳转 `/library` |
| 查看全部 | 最近使用跳转 `/library`，收藏分类跳转 `/category` |
| 最近使用项 | 点击进入详情页 |
| 星标按钮 | 调用 `toggleFavorite()`，保留当前列表位置 |
| 更多按钮 | 打开现有提示词更多菜单 |
| 分类卡片 | 跳转提示词库并带 `folder` 参数 |
| 新建提示词 | 跳转 `/editor/` |
| 导入提示词 | 打开 JSON 文件选择并调用现有导入逻辑 |

## 7. 动效与响应式

- 统计卡和最近使用项使用 0.2s 左右的轻量淡入上移。
- 首页欢迎横幅中部红框区域使用共享像素柯基 `steps()` 序列帧动画，并以 CSS 变量控制帧数与节奏；动画层 `pointer-events: none`，不影响页面操作。
- 卡片悬停只做轻微上移和阴影增强，不改变布局尺寸。
- 星标、更多按钮都有 `focus-visible` 状态；搜索框依赖输入框原生聚焦状态。
- `prefers-reduced-motion: reduce` 下关闭首页动效、像素序列帧和横向位移动效，仅保留静态第一帧。
- PC 默认窗口按 `1366 x 768` 的 `16:9` 桌面比例设计，页面使用 PC 安全内边距变量，避免默认窗口下内容被侧边栏和固定留白挤压。
- PC 左侧侧边栏支持展开与收起，收拉按钮位于原安全徽章区域；展开态为满宽胶囊按钮并显示文案，收起态为窄图标按钮；首页不读取侧边栏状态，而是依赖 `.pc-main` 容器查询随主内容区宽度自然缩放。
- 欢迎横幅背景层按 `.pc-main` 主内容可视宽度铺满，内部标题、说明和吉祥物仍按页面安全宽度居中，避免侧边栏收起或宽屏场景下背景图只覆盖固定内容容器。
- 像素动画只在主内容区宽度充足时展示；约 1180px 以下隐藏，优先保护标题、副标题、搜索框和右侧插画不重叠。
- 首页搜索框使用轻量负外边距贴近横幅下沿，不再大幅上拉压住欢迎区；搜索、统计卡、最近使用和底部双栏之间保持约 18px 到 20px 的稳定节奏。
- 约 1180px 以下最近使用退化为单列，底部双栏退化为上下堆叠。
- 当主内容区变窄时，统计卡、收藏分类和快捷创建优先从四列退化为两列，再由底部模块纵向堆叠，保持卡片标题、数字和按钮不重叠。
- 最近使用空状态始终跨满列表区域，图标与文本在面板内自适应居中，文本自动换行且不溢出。
- 约 980px 以下统计卡改为两列，收藏分类改为两列。

## 8. 验收标准

- [ ] 首页首屏结构符合参考图：欢迎横幅、悬浮搜索、四张统计卡、最近使用和底部双栏。
- [ ] 首页红框内容区整体位置舒展，搜索框不压迫欢迎副标题和右侧插画。
- [ ] 首页红框区域显示像素柯基序列帧动画，背景透明且不出现棋盘格。
- [ ] 窄宽和减弱动态场景下像素动画可隐藏或静止，不造成标题、插画和搜索框重叠。
- [ ] 统计卡显示正确数量，并包含图标、标题和数字。
- [ ] 最近使用最多展示 6 条，按更新时间倒序，缩略图优先显示真实图片。
- [ ] 最近使用无数据或搜索无结果时，空状态图标与文本始终在列表区域内居中显示。
- [ ] 收藏分类最多展示 4 个，并显示分类名称和数量。
- [ ] 搜索、查看全部、详情跳转、收藏切换、更多菜单、分类跳转、新建和导入均可用。
- [ ] 左侧侧边栏展开与收起后，首页主内容区均可自适应重排且无文字挤压。
- [ ] 1024px、1280px、1440px 桌面宽度下无文字重叠、无横向滚动、无布局抖动。
- [ ] 动效尊重 `prefers-reduced-motion`。
