# 界面1：首页仪表盘

> 版本：1.0 | 最后更新：2026-05-05

---

## 1. 页面定位与目标

首页是用户进入应用后的仪表盘，用于：
- 快速了解提示词管理全貌（统计数据）
- 查看最近使用的提示词
- 查看收藏分类
- 快速创建新提示词或导入提示词

**核心目标**：让用户一眼掌握全局，一键进入操作。

| 元素 | 颜色 | 说明 |

## 5. 数据绑定

| 数据 | 来源 | API/方法 |
|------|------|---------|
| 提示词总数 | 聚合查询 | `getPromptSets().length` |
| 分类数 | 聚合查询 | `getFolders().length` |
| 标签数 | 聚合查询 | 使用现有标签数据 |
| 最近使用列表 | 按更新时间排序 | `getPromptSets()` 排序取前6个 |
| 收藏分类列表 | 文件夹+数量 | `getFolders()` + 各文件夹下集合数 |

## 6. 交互规范

### 6.1 交互状态矩阵

| 元素 | 默认态 | Hover 态 | 选中态 |
|------|--------|---------|--------|
| 搜索框 | `--surface2` 背景 | 边框→`--accent` | — |
| 统计卡片 | 浅色背景 | 边框→`--accent` | — |
| 最近使用项 | 透明背景 | 背景→`--surface2` | 背景→`--accent-bg` + 边框→`--accent` |
| 收藏星标 | 灰色轮廓 | — | `--accent` 填充 |
| 三点菜单 | 灰色图标 | 图标→`--text` | — |
| 收藏分类卡片 | 彩色背景 | 边框→`--accent` | — |
| 新建按钮 | `--accent` 背景 | 背景→`--accent2` | — |

### 6.2 操作流程

```
点击「新建」 → 侧边栏新增一项 → 自动选中 → 右侧显示详情
                               ↓
                          Toast: 「已创建新提示词集合」
```

### 6.3 动画过渡

| 动画 | 时长 | 说明 |
|------|------|------|
| 页面加载 | 0.25s | 淡入 |
| 统计卡片加载 | 0.25s | 依次淡入 |
| 列表项 Hover | 0.15s | 背景色渐变 |
