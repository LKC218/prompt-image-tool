# 界面4：新建/编辑提示词

> 版本：1.0 | 最后更新：2026-05-10

---

## 1. 页面定位与目标

用于新建或编辑提示词，包括：
- 输入提示词集合名称
- 输入正向/负向提示词
- 选择分类
- 管理版本

**核心目标**：提供清晰高效的编辑体验。

---

## 2. 页面结构

页面顶部使用共享欢迎横幅组件，新建和编辑状态展示不同标题与副标题；布局参考顶部横幅设计稿，左侧依次为返回按钮、标题和副标题，右侧为柯基插画与保存按钮。横幅外层与提示词库一致撑满主内容区，表单主体保持居中双栏编辑布局，底部不追加独立的本地保存提示条。

## 5. 数据绑定

| 数据 | 来源 | 说明 |
|------|------|------|
| 标题 | `promptSet.name` | 新建时为空 |
| 分类 | `promptSet.folder_id` | 新建时为 null |
| 正向提示词 | `version.positive_prompt` | 新建时为空 |
| 负向提示词 | `version.negative_prompt` | 新建时为空 |
| 版本列表 | `promptSet.versions` | 新建时只有 v1 |

### 创建逻辑

```
点击「创建」
├── 校验：标题不为空
├── 创建提示词集合 → createPromptSet()
├── 创建版本 → createVersion()
├── Toast: 「已创建」
└── 切换到详情页
```

## 6. 交互规范

### 6.1 交互状态矩阵

| 元素 | 默认态 | Hover 态 | 焦点态 |
|------|--------|---------|--------|
| 输入框 | `--surface2` 背景 | — | `--accent` 边框 + `--accent-bg` 外阴影 |
| 创建按钮 | `--accent` 背景 | `--accent2` 背景 | — |
| 分类选择器 | `--surface2` 背景 | — | `--accent` 边框 |
| 返回按钮 | `--surface2` 背景 | `--accent-bg` 背景 | — |

### 6.2 操作流程

```
点击「返回」→ 弹出确认弹窗 → 确认 → 返回 → 确认放弃更改 → 返回列表
```

### 6.3 动画过渡

| 动画 | 时长 | 说明 |
|------|------|------|
| 输入框焦点 | 0.15s | 边框+外阴影渐变 |
| 创建按钮 Hover | 0.15s | 背景色渐变 |
| 弹窗显隐 | 0.25s | 淡入淡出+缩放 |
