# 界面4：新建/编辑提示词

> 版本：1.0 | 最后更新：2026-05-10

---

## 1. 页面定位与目标

用于新建或编辑提示词，包括：
- 输入提示词集合名称
- 输入正向/负向提示词
- 选择分类
- 管理版本

**核心目标**：提供清晰高效的编辑体验。

---

## 2. 页面结构

页面顶部使用共享欢迎横幅组件，新建和编辑状态展示不同标题与副标题；布局参考顶部横幅设计稿，左侧依次为返回按钮、标题和副标题，右侧为柯基插画与保存按钮。新建模式在保存按钮左侧提供低权重“一键清空”按钮，用于清除当前未保存输入、图片预览和新建草稿；编辑模式不显示该按钮，避免误清已保存内容。横幅通过 `renderPcWelcomeWalkAnimation({ variant: 'editor' })` 接入保守走路动画轨道，空间不足时优先隐藏以避开保存动作区。横幅外层与提示词库一致撑满主内容区，表单主体保持居中双栏编辑布局，底部不追加独立的本地保存提示条。

## 3. 视觉规范

### 3.1 暖白拟态层级

页面统一使用左上浅色高光、右下暖棕暗影。编辑器主卡、上传入口、分类选择器、比例按钮和次级操作默认外凸；标题和提示词输入区、图片区及已选比例使用内凹阴影。组件阴影统一复用 `--pc-neu-*` 令牌，主题色仅用于焦点、悬停和已选状态。

| 元素 | 默认态 | 交互态 |
|------|--------|--------|
| 保存按钮 | 品牌色高对比主按钮 | 保持既有保存中反馈，不采用低对比拟态 |
| 编辑器主卡 | 暖白外凸工作台 | 无独立按压态 |
| 标题与提示词输入 | 暖白内凹输入槽 | `--pc-accent` 焦点边框、内凹主题阴影与 `--pc-accent-bg` 焦点环 |
| 图片区 | 内凹图片区底槽 | 上传入口、缩略图和追加入口外凸，按压时内凹 |
| 分类与比例 | 默认外凸 | 已选或按压内凹，并以 `--pc-accent` 标识状态 |
| 清空与添加标签 | 轻量外凸次级按钮 | 悬停强化双向阴影，按压切换内凹 |
| 图片删除 | 深色高对比危险浮层 | 保持与图片背景的可见性，不转换为暖白拟态 |

图片区已有图片缩略图使用放大镜指针，点击或键盘 Enter/Space 打开公共图片查看器；删除按钮独立处理事件，不会触发查看器。空图片区和“添加图片”入口继续用于上传。

### 3.2 尺寸规范

| 属性 | 值 |
|------|-----|
| 标题输入框字号 | 16px |
| 提示词输入框最小高度 | 140px |
| 输入框圆角 | `--pc-radius-md` (16px) |
| 输入框内边距 | 14px 16px |

```

```

## 6. 交互规范

### 6.1 交互状态矩阵

| 元素 | 默认态 | Hover 态 | 焦点态 |
|------|--------|---------|--------|
| 输入框 | 内凹暖白输入槽 | 保持内凹层级 | 主题色边框与可见焦点环 |
| 创建按钮 | 高对比品牌主按钮 | 维持既有品牌动效 | — |
| 新建页一键清空按钮 | 外凸暖白次级按钮 | 阴影强化并上移 1px | 主题色外描边 |
| 分类选择器 | 外凸暖白选择器 | 阴影强化并上移 1px | 主题色外描边，原生按钮语义 |
| 比例按钮 | 外凸暖白分段按钮 | 阴影强化并上移 1px | 已选态内凹且同步 `aria-pressed` |
| 图片上传入口 | 外凸暖白上传按钮 | 阴影强化并上移 1px | 主题色外描边 |
| 图片缩略图 | 外凸暖白相框与放大镜指针 | 阴影强化并上移 1px | Enter/Space 打开公共图片查看器 |
| 文本选区菜单 | 选区稳定后自动显示复制、粘贴、删除菜单 | 锚定真实选区，优先上方并留出 20px，菜单不夺取输入焦点；复制、粘贴、删除使用柔和语义色 | Esc 关闭并保留字段焦点与选区 |

PC 精细指针设备启用全局主题色自定义光标：中心点即时跟随，外环延迟追随；进入按钮、导航和可聚焦操作控件时外环放大。非语义点击容器必须添加 `data-cursor="action"`，图片缩略图、拖拽、缩放、禁用和加载等保留原生指针的区域必须添加 `data-cursor="native"`。标题、提示词等文本输入控件保留原生文本插入光标；触控设备或系统开启“减少动态效果”时不启用该效果。

### 6.2 操作流程

```
点击「清空」→ 弹出确认弹窗 → 确认 → 清空标题、提示词、标签、分类、比例、图片预览和新建草稿
点击图片缩略图或按 Enter/Space → 打开图片查看器；点击删除按钮 → 删除对应图片且不打开查看器
标题、正向或负向提示词选中文本后 → 约 180ms 在真实选区上方自动显示复制、粘贴、删除菜单，间距为 20px；上方空间不足时切换到下方，菜单与柔焦层距离视口边缘至少 24px，同时保持输入框焦点与主题色选区高亮。复制、粘贴和删除悬浮时分别使用青绿、蓝紫和珊瑚红语义色；取消选区、继续输入、输入法组合或点击外部 → 关闭菜单
```

### 6.3 分类选择弹窗

分类选择弹窗使用浮层表面色与细边框，列表容器使用次级表面色。列表默认文字和图标使用次级文字色，悬停态切换为抬升表面与细边框，选中态使用低饱和品牌背景、品牌色文字及勾选标记。暗色模式不得使用固定纯黑图标或固定浅粉选中底色；文件夹、右箭头、勾选、关闭和添加标签加号均使用单一 CSS 蒙版与语义 Token 着色，不保留外部 SVG 图片节点。弹窗支持点击遮罩、关闭按钮与 Escape 关闭；关闭或选中后焦点回到分类选择器。

### 6.4 动画过渡

| 动画 | 时长 | 说明 |
|------|------|------|
| 拟态控件状态 | `--pc-neu-transition` | 阴影、颜色与 1px 位移过渡 |
| 输入框焦点 | `--pc-neu-transition` | 主题色边框、内凹主题阴影与焦点环 |
| 创建按钮 Hover | 0.15s | 维持既有品牌动效 |
| 弹窗显隐 | 0.25s | 淡入淡出+缩放 |

| 本页面功能 | 对应现有功能 | 说明 |
|-----------|-------------|------|

| 创建按钮 | 现有侧边栏「新建」+自动创建 | 显式创建 |

## 8. 比例与响应式补充

- PC 默认窗口采用 `1366 x 768`，编辑页在中等宽度下提前从主表单加侧栏退化为单栏，避免分类、比例和插画区域压缩主输入区。
- 主表单和侧栏卡片均设置抗挤压基础规则，文本域、标签、比例按钮使用整组换行而不是按钮内部逐字换行。
- 比例选项包含 `1:1`、`3:4`、`4:3`、`16:9`、`9:16` 和 `21:9`；比例值沿用 `aspectRatio` / `aspect_ratio` 既有兼容链路保存、恢复和导入。
- 图片预览网格使用自适应列宽，上传、删除和查看按钮保持固定交互尺寸，避免图片加载前后布局跳动；图片区支持点击上传和 `Ctrl+V` 粘贴外部图片，粘贴图片追加到当前图片末尾。
- 新建和编辑页通过本地草稿记录未保存内容，返回其他页面再回来时恢复标题、提示词、标签、分类、比例和新导入图片；正式保存成功后清理对应草稿。
- 新建模式一键清空会先二次确认，确认后删除 `pc-editor-draft:new`，避免清空后切页回来再次恢复旧输入；清空范围包含点击上传、剪贴板粘贴和 prompt-image-tool JSON 导入预填产生的图片。
- 更窄窗口下隐藏编辑页插画，将空间优先留给标题、提示词和保存操作。
