# 界面6：设置与本地存储

> 版本：1.0 | 最后更新：2026-05-05

---

## 1. 页面定位与目标

展示应用设置和本地存储信息，强调本地安全：
- 外观设置（深色模式/浅色模式）
- 本地存储数据查看
- 数据备份与恢复
- 局域网同步
- 本地安全提示

**核心目标**：让用户安心使用，明确数据安全，提供数据管理入口。

## 6. 交互规范

### 6.1 交互状态矩阵

| 元素 | 默认态 | Hover 态 | 选中态 |
|------|--------|---------|--------|
| 深色模式开关 | 灰色轨道 | — | 紫色轨道+白色滑块 |
| 数据管理按钮 | `--surface2` 背景 | `--accent-bg` 背景 + `--accent` 边框 | — |
| 局域网同步按钮 | `--surface2` 背景 | `--accent-bg` 背景 + `--accent` 边框 | — |

### 6.2 操作流程

```
点击深色模式开关 → 切换主题 → 更新 localStorage → 触发全局样式
```

```
点击「导出数据」 → 触发文件下载 → Toast: 「已导出」
```

```
点击「导入数据」 → 触发文件选择 → 选择文件 → 导入 → Toast: 「已导入」
```

```
点击「同步设置」 → 打开局域网同步弹窗 → 输入 IP → 开始同步
```

### 6.3 动画过渡

| 动画 | 时长 | 说明 |
|------|------|------|
| 主题切换 | 0.25s | 背景色+文字色渐变 |
| 开关切换 | 0.2s | 滑块滑动 |
| 按钮 Hover | 0.15s | 背景色+边框渐变 |
