# 界面5：分类管理

> 版本：1.0 | 最后更新：2026-05-05

---

## 1. 页面定位与目标

用于管理分类，包括：
- 查看所有分类
- 新建分类
- 重命名分类
- 更改分类颜色
- 删除分类
- 移动提示词到分类

**核心目标**：让用户高效组织提示词的分类体系。

## 6. 交互规范

### 6.1 交互状态矩阵

| 元素 | 默认态 | Hover 态 | 选中态 |
|------|--------|---------|--------|
| 分类项 | `--surface` 背景 | `--surface2` 背景 | — |
| 三点菜单 | 灰色图标 | 图标→`--text` | — |
| 新建分类按钮 | `--accent` 背景 | `--accent2` 背景 | — |

### 6.2 操作流程

```
点击「+ 新建分类」 → 弹出弹窗 → 输入名称+选择颜色 → 创建 → Toast: 「已创建」
```

```
右键分类项 → 弹出右键菜单 → 选择操作（重命名/更改颜色/删除）
```

```
右键提示词集合 → 弹出右键菜单 → 选择「移动到文件夹」 → 选择目标分类 → 移动 → Toast: 「已移动」
```

### 6.3 动画过渡

| 动画 | 时长 | 说明 |
|------|------|------|
| 分类项 Hover | 0.15s | 背景色渐变 |
| 弹窗显隐 | 0.25s | 淡入淡出+缩放 |
| 分类项删除 | 0.25s | 向左滑出+高度收缩 |
