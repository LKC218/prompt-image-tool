# 界面3：提示词详情

> 版本：1.0 | 最后更新：2026-05-05

---

## 1. 页面定位与目标

详情页用于查看和编辑单条提示词，包括：
- 查看和编辑提示词集合名称
- 管理版本（切换/新增/删除/重命名/复制）
- 查看和编辑正向/负向提示词
- 查看和编辑版本备注
- 管理图片（上传/删除/查看）
- 版本对比

**核心目标**：让用户快速编辑和管理提示词内容。

## 3. 视觉规范

### 3.1 页面专用颜色

| 元素 | 颜色 | 说明 |
|------|------|------|
| 详情 Header 背景 | `--surface` | 表面色 |
| 版本标签-选中 | `--accent` 背景 + 白色字 | 紫色 |
| 版本标签-默认 | `--surface2` 背景 + `--text2` 文字 | 灰色 |
| 版本标签 Hover | `--accent-bg` 背景 + `--accent` 边框 | 紫色高亮 |
| 最新版本徽章 | `--badge-success-bg` 背景 + `--success` 文字 | 绿色 |
| 历史版本徽章 | `--badge-warning-bg` 背景 + `--warning` 文字 | 黄色 |
| 操作按钮 | `--surface2` 背景 | 灰色 |
| 复制按钮 | 透明背景 + `--accent` 边框 + `--accent` 文字 | 紫色 |

## 5. 数据绑定

| 数据 | 来源 | API/方法 |
|------|------|---------|
| 集合名称 | `promptSet.name` | 可编辑 |
| 当前版本 | `promptSet.versions[activeIndex]` | 可切换 |
| 正向提示词 | `version.positive_prompt` | 可编辑 |
| 负向提示词 | `version.negative_prompt` | 可编辑 |
| 版本备注 | `version.note` | 可编辑 |
| 图片列表 | `version.images` | 可上传/删除 |
| 所有版本 | `promptSet.versions` | 可管理 |

## 6. 交互规范

### 6.1 交互状态矩阵

| 元素 | 默认态 | Hover 态 | 焦点态 |
|------|--------|---------|--------|
| 集合名称输入框 | 透明背景+无边框 | — | 底部边框→`--accent` |
| 版本标签 | 灰色背景+灰色文字 | 紫色背景+紫色边框 | 紫色背景+白色文字 |
| 提示词文本域 | `--surface2` 背景 | — | 边框→`--accent` |
| 复制按钮 | 透明背景+紫色边框 | 紫色背景+白色文字 | — |
| 图片卡片 | `--surface2` 背景 | 边框→`--accent` + `--shadow-md` | — |
| 添加图片区 | 虚线边框 | 虚线边框→`--accent` | — |

### 6.2 操作流程

```
编辑提示词 → 输入 → 500ms 防抖 → 自动保存
                            ↓
                       无视觉反馈
```

```
点击「+ 新版本」 → 弹出新建版本弹窗 → 输入版本号 → 创建 → Toast: 「已创建新版本」
```

```
点击「版本对比」 → 展开对比面板 → 选择左右版本 → 实时更新对比内容
```

### 6.3 动画过渡

| 动画 | 时长 | 说明 |
|------|------|------|
| 版本切换 | 0.25s | 淡入切换 |
| 提示词输入 | 实时 | 自动保存无视觉反馈 |
| 图片上传 | 0.25s | 淡入显示 |
| 对比面板展开/收起 | 0.25s | 高度动画 |
