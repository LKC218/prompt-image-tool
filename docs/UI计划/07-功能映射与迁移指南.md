## 二、数据模型对照

### 2.1 现有数据模型（SQLite）

```sql
-- 提示词集合表
prompt_sets (
    id TEXT PRIMARY KEY,
    name TEXT,
    folder_id TEXT,
    created_at TEXT,
    updated_at TEXT
)

-- 版本表
versions (
    id TEXT PRIMARY KEY,
    prompt_set_id TEXT,
    name TEXT,
    note TEXT,
    positive_prompt TEXT,
    negative_prompt TEXT,
    created_at TEXT,
    order_num INTEGER
)

-- 图片表
images (
    id TEXT PRIMARY KEY,
    version_id TEXT,
    file_path TEXT,
    note TEXT,
    order_num INTEGER
)

-- 文件夹表
folders (
    id TEXT PRIMARY KEY,
    name TEXT,
    color TEXT,
    order_num INTEGER
)
```

### 2.2 移动端新增字段（Phase 1 仅 UI 展示，不持久化）

| 表 | 新增字段 | 类型 | 默认值 | 说明 |
|----|---------|------|--------|------|
| prompt_sets | `is_favorite` | INTEGER | 0 | 收藏标记 |
| prompt_sets | `tags` | TEXT | '[]' | 标签JSON数组 |
| prompt_sets | `cover_image` | TEXT | null | 封面图路径 |
| versions | `aspect_ratio` | TEXT | '1:1' | 比例 |
| versions | `style_preset` | TEXT | null | 风格预设 |

### 2.3 数据模型扩展计划

| Phase | 扩展内容 | 说明 |
|-------|---------|------|
| Phase 1 | 仅 UI 展示，使用 mock 数据 | 收藏/标签/比例/风格在界面中展示，不持久化 |
| Phase 2 | 扩展 SQLite 表结构 | 新增字段，编写数据迁移脚本 |
| Phase 3 | 完整功能闭环 | 收藏筛选、标签筛选、数据导入导出兼容 |

## 三、交互方式迁移

### 3.1 触发方式对照

| 现有交互 | PC端触发 | → 移动端触发 | 说明 |
|---------|---------|-------------|------|
| 右键菜单 | 鼠标右键 | 三点菜单按钮 | 移动端无右键，改为显式按钮 |
| 长按菜单 | 500ms长按 | 三点菜单按钮 | 统一为三点菜单，更直观 |
| 拖拽图片 | 拖拽到添加区域 | 点击上传按钮 | 移动端无拖拽，改为按钮触发 |
| Hover 显示删除按钮 | 鼠标悬停 | 滑动删除或编辑模式 | 移动端无 Hover |
| 双列版本对比 | 并排显示 | 上下堆叠或Tab切换 | 屏幕宽度限制 |
| 侧边栏切换 | 列表/文件夹视图按钮 | 底部导航切换页面 | 导航结构变化 |

### 3.2 手势操作规范

| 手势 | 操作 | 使用场景 |
|------|------|---------|
| 点击 | 选中/打开 | 列表项、按钮、标签 |
| 长按 | 进入编辑模式 | 分类列表拖拽排序触发 |
| 左滑 | 删除/操作 | 列表项快捷操作（可选） |
| 下拉 | 刷新 | 列表页刷新数据 |
| 右滑（边缘） | 返回 | 全局返回手势 |

## 四、页面路由映射

### 4.1 路由表

| 路由路径 | 页面 | 底部导航高亮 | 对应现有功能 |
|---------|------|-------------|-------------|
| `/` | 首页仪表盘 | 首页 | 侧边栏列表 + 统计 |
| `/library` | 提示词库 | 提示词库 | 侧边栏列表 + 搜索 |
| `/detail/:id` | 提示词详情 | 提示词库 | 详情页 |
| `/editor/:id?` | 新建/编辑提示词 | — | 详情页编辑模式 |
| `/category` | 分类与标签 | 分类 | 文件夹管理 |
| `/settings` | 设置与本地存储 | 设置 | Header功能集合 |

### 4.2 页面层级关系

```
首页 (/)
├── 提示词库 (/library)
│   ├── 提示词详情 (/detail/:id)
│   │   ├── 编辑提示词 (/editor/:id)
│   │   └── 版本对比 (模态)
│   └── 新建提示词 (/editor/)
├── 分类与标签 (/category)
│   └── 编辑分类 (模态)
└── 设置 (/settings)
    ├── 局域网同步 (模态)
    └── 帮助文档 (模态)
```
