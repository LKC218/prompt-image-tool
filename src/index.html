使用帮助

提示词管理
点击侧边栏「+ 新建」创建提示词集合。选中集合后可编辑正向提示词和反向提示词，内容会自动保存。支持复制、删除集合，也可右键/长按进行更多操作。
版本管理
每个集合支持多个版本，方便迭代和对比。点击版本标签切换版本，点击「+ 新版本」添加新版本。支持版本重命名、删除、复制为新版本。点击「版本对比」可并排查看两个版本的提示词和图片差异。
图片管理
在版本详情中，点击图片区域或拖拽图片即可上传参考图。每张图片可添加备注，点击图片可全屏查看。支持删除单张图片。
文件夹分类
点击侧边栏底部的「+ 新建文件夹」创建分类文件夹，可为文件夹选择颜色标签。右键/长按集合项可选择移动到指定文件夹。点击文件夹可筛选显示对应集合。
数据导入/导出
点击顶部「导出」按钮将所有数据导出为 JSON 文件。点击「导入」按钮选择 JSON 文件恢复数据。导入时会与现有数据合并，不会覆盖。
局域网同步
在手机端点击顶部「同步」按钮，输入 PC 端显示的 IP 地址，即可将手机端数据同步到电脑。请确保两端处于同一局域网，且 PC 端软件已启动。同步为单向操作（手机→电脑），冲突数据会自动保留副本。
主题切换
点击顶部的太阳/月亮图标可在暗色和亮色主题之间切换。应用会自动检测系统主题偏好。
手势操作
在首页使用返回手势或返回键时，会提示「再按一次退出应用」，2秒内再次操作才会退出，防止误触。在详情页等其他页面，返回操作会回到上一级。
